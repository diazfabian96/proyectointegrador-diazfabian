Argentina Programa

Lleve adelante una aplicación web full stack, que muestra: mis datos personales, estudios cursados, experiencia laboral, conocimiento de las tecnologías entre otras cosas. Es decir, un propio portfolio web. Esta aplicación es de arquitectura distribuida y contiene un diseño de interfaz de usuario (front end) que muestra la información, una base de datos que almacena los datos antes mencionados y cuenta con las APIs necesarias para proveer a través de internet la información (back end).El portfolio web esta diseñado como carta de presentación para el mercado laboral.
